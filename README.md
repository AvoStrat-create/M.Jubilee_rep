# This project is a Python-based simulation of the BB84 quantum cryptography protocol, which demonstrates how two users (Alice and Bob) can securely share a secret key using quantum principles. In the simulation, Alice generates random bits and encodes them using randomly chosen quantum bases, while Bob measures them using his own random bases. Because of the rules of quantum mechanics, only the bits where Alice and Bob use the same basis are kept to form a shared secret key. The project also includes an eavesdropper (Eve) who tries to intercept the communication, and her interference introduces errors that can be detected by comparing parts of the key. This shows how quantum cryptography can ensure secure communication and detect hacking attempts.
